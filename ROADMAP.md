# Roadmap

- Core evaluation + persistence + cache
- Middleware, Blade, Artisan
- Optional management API
- Audit log
- Test suite + CI

## v0.2

- Filament 3 resources for flag management
- `active_from` / `active_until` schedules
- JSON import/export

## v0.3

- Feature dependencies (flag requires other flags)
- Metrics hooks (evaluation counters)
- Vue/Inertia composable

## v1.0

- Experiments (A/B variants)
- Stable public API guarantees
- Example SaaS app under `examples/`

Items above are planned, not shipped, until released.
