# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) spirit:

- Be respectful and inclusive.
- Assume good intent; disagree on ideas, not people.
- No harassment, spam, or personal attacks.
- Maintainers may moderate issues, PRs, and discussions accordingly.

Report conduct concerns privately to the maintainer.
